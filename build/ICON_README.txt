# Icon File

Place your application icon file here as:
- `icon.ico` - Windows icon (256x256 pixels recommended)

You can create an ICO file from a PNG using online tools or:
- https://www.icoconverter.com/
- https://convertio.co/png-ico/

Recommended sizes to include in ICO:
- 16x16
- 32x32
- 48x48
- 256x256

If no icon is provided, the default Electron icon will be used.
